# Overview
Storage benchmarking of VMware ESXi cluster using diskspd and PowerShell PowerCLI.

# How to use?
Step 1: Provide details of your setup in the configuration data file (benchmarking_manifest.psd1) <br />
Step 2: Invoke deploy_test_vms.ps1 for creating stress test VMs on all ESXi nodes in the cluster <br />
